# 布局

## 基本介绍

宽度划分组件  
使用时在`Layout`组件中添加`div`标签, `div`标签中填写`width`属性即可按比例划分出区域

## Example&Format

```javascript
<Layout>
    <div width="20">test1</div>
    <div width="20"></div>
    <div width="40">
        test2
        <div>test2.1</div>
        <div>test2.2</div>
        <div>test2.3</div>
    </div>
    <div width="20">test3</div>
</Layout>
```
